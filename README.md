# ApexCoreFitnessWebsite


## Project Description
Apex Core Fitness is a multi-page fitness website created for my Web Development course.
The site includes a homepage, workout programs, guides, store, and a resources page. 
It demonstrates HTML and CSS skills. 

##Installation Instructions
No installation necessary
Project will run in any modern browser

##Usage Instructions
Use the navigation at the top of each page to explore the site:
**Home** - Overview of Apex Core Fitness
**Programs** - Workout programs
**Guides** - Fitness guides and tips
**Store** - Products and merchandise
**Resources** - Additional information

All pages linked together for easy navigation
